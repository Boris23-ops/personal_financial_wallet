# Личный финансовый кошелек


### Описание проекта
Приложение представляет собой программу для ведения записей о доходах и расходах. Пользователь может добавлять новые записи, редактировать или удалять существующие, а также осуществлять поиск по критериям. Приложение позволяет загружать данные из файла, сохранять изменения и выводить текущий баланс, доходы и расходы.

### Структура классов
1. Record - Класс, представляющий запись о финансовой операции.

- Поля: date - дата операции, category - категория (Доход/Расход), amount - сумма, description - описание операции.

2. RecordManager - Класс, управляющий работой с записями.

- Поля: records - список всех записей.

- Методы:
- load_records(filename: str) - загружает записи из файла.
- save_records(filename: str) - сохраняет текущие записи в файл.
- display_balance() - отображает текущий баланс, доходы и расходы.
- add_record(new_record: Record) - добавляет новую запись.
- edit_record(index: int, new_record: Record) - редактирует существующую запись по индексу.
- search_records(category: str, date: str, amount: int) - ищет записи по заданным критериям.

### Интерфейс пользователя
1. Главное меню

- Вывод баланса

- Добавление записи

- Редактирование записи

- Поиск по записям

2. Функционал

- Ввод данных для новой записи: дата, категория, сумма, описание.

- Редактирование записи: выбор записи по индексу, ввод новых значений полей.

- Поиск по записям: возможность указать категорию, дату и сумму для поиска.

### Запуск приложения
1. Программа начинает работу с загрузки данных из файла "data.txt".
2. Пользователь выбирает действие из основного меню.
3. В зависимости от выбора, запускается соответствующий функционал.
4. Для сохранения изменений внесенных в записи, данные автоматически сохраняются в файл.
